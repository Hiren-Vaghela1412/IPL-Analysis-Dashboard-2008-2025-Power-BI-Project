## 📌 Project Overview

This project presents an interactive Power BI dashboard built using IPL (Indian Premier League) data from 2008 to 2025.

The dashboard provides comprehensive insights into team performance, season statistics, batting records, bowling records, tournament winners, Orange Cap and Purple Cap holders, and overall IPL trends through interactive visualizations.

---

## 🚀 Features

### Season Analysis
- Season Winner
- Season Runner-Up
- Season-wise Filtering

### Tournament Statistics
- Total Matches
- Total Teams
- Total Venues
- Total Fours
- Total Sixes
- Total Centuries
- Total Half-Centuries

### Player Insights
- Orange Cap Holder
- Purple Cap Holder
- Most Fours
- Most Sixes

### Team Performance
- Points Table
- Matches Played
- Wins
- Losses
- No Results
- Tie Matches
- Total Points

## 📈 Key Insights

- Analyze season-wise IPL performance.
- Compare team standings across seasons.
- Track Orange Cap and Purple Cap winners.
- Identify players with the most runs, wickets, fours, and sixes.
- Explore batting and bowling trends across IPL history.
- Examine team consistency through points table analysis.
